Description:
------------
Kiosk is a Windows utility for launching a web browser in kiosk mode.
All other browsers are closed, secondary monitors can be disabled, and the Windows taskbar is hidden for maximum focus.

How to Use:
-----------
1. Run Kiosk.exe.
2. Select your preferred browser.
3. Enter your desired website URL (e.g., www.chess.com).
   The program will automatically add the "https://" prefix if you forget.
4. Optionally, check the "Disable secondary monitors" box.
5. Click "ENTER KIOSK MODE."

Exiting Kiosk Mode:
-------------------
Press Ctrl + Alt + Q at any time to exit kiosk mode, restore your taskbar, and re-enable monitors.

Important Notes:
----------------
- All open browser sessions are closed when kiosk mode starts.
- Please save your work before launching KioskAI.
